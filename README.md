# Calentra

Calentra is a full-stack appointment scheduling and calendar management system designed for small service-based businesses.

The system replaces manual appointment books with a structured, digital workflow, focusing on clean backend architecture, data modeling, and real-world business logic.

---

## Key Features (Planned)

- Secure authentication system (password hashing, JWT, protected routes)
- Customer management (contact details, notes, WhatsApp opt-in)
- Treatment catalog with duration and pricing
- Appointment scheduling with calendar views:
  - Daily
  - Weekly
  - Monthly
- Appointment lifecycle management:
  - Scheduled
  - Completed
  - Canceled
  - No-show
- Customer profile with full appointment history
- Automated WhatsApp appointment reminders
- Conflict prevention for overlapping appointments
